# Future TL Weather Overlay

## Для пользователей

Запустите один из файлов из папки `release`:

- `Future-TL-Weather-Overlay-Portable-0.1.2.exe` — portable-версия, установка не нужна.
- `Future-TL-Weather-Overlay-Setup-0.1.2.exe` — установщик Windows.

## Управление

- `Ctrl+Shift+W` — открыть или закрыть настройки.
- `Ctrl+Shift+H` — скрыть или показать overlay.
- В режиме настроек ленту можно перетащить мышью.
- Когда настройки закрыты, overlay работает в click-through режиме и не мешает управлению в игре.
- Если игра уже запускалась и процесс `TL` завершился, overlay полностью закрывается автоматически.
- Если доступна новая версия, в настройках появится кнопка `ОБНОВИТЬ`.
- После нажатия `ОБНОВИТЬ` overlay скачает обновление, установит его тихо и перезапустится.

## Важно

Overlay является отдельным приложением. Оно не изменяет файлы игры, не читает память игры, не внедряется в процесс игры и не автоматизирует ввод.

Если Windows SmartScreen предупреждает о неизвестном издателе, причина в том, что приложение не подписано коммерческим code-signing сертификатом.
